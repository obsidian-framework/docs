# Installation

## Prerequisites

- Java 17 or higher
- Maven 3.6+
- MySQL / PostgreSQL / SQLite

## Getting Started

**1. Clone the skeleton**

```bash
git clone https://github.com/kainovaii/obsidian-skeleton.git
cd obsidian-skeleton
```

**2. Run the application**

```bash
./build.bat
```

That's it. The application will start and scan all annotated components automatically.
